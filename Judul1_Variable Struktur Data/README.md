**Tugas Akhir Judul 1**

**Program Daftar Tugas Menggunakan List**

**Deskripsi Singkat**

Program ini merupakan program sederhana yang digunakan untuk mengelola daftar tugas. Program ini memiliki menu utama seperti menampilkan tugas, menambah tugas, dan menghapus tugas. Data tugas disimpan dalam struktur data list pada python untuk menyimpan kumpulan tugas. Program juga memiliki validasi inputan untuk mengurangi kesalahan. Program ini juga menggunakan perulangan while agar program dapat terus berjalan samapai user memilih pilihan keluar dari program.  

**Source Code**

<img width="1356" height="2610" alt="SS tugas akhir 1" src="https://github.com/user-attachments/assets/3c079430-7c80-4ab5-9a89-4b577ba88693" />


Pada baris 1-6 mendefinisikan fungsi menu() untuk menampilkan daftar pilihan kepada user, seperti menampilkan tugas, menambahkan tugas, menghapus tugas, dan keluar program. 

Baris 8-10 terdapat fungsi main () sebagai fungsi utama dan list kosong untuk menyimpan tugas. Lalu running yang digunakan agar program berjalan jika nilainya true.

Baris 12-16 program menggunakan perulangan while agar terus menampilkan menu dan membaca input dari user. Terdapat juga untuk memastikan bahwa inputan berupa angka.

Pada baris 20-26, jika user memilih pilihan satu maka program akan mengecek apakah list kosong atau menampilkan daftar tugas. Baris 28-34, jika memilih pilihan dua maka akan ada opsi untuk menambahkan tugas baru.

Selanjutnya baris ke 36-50 merupakan pilihan ketiga yaitu pilihan untuk menghapus tugas menggunakan pop().

Pada baris 52-55 merupakan pilihan keluar yang akan menghentikan program dengan mengubah true menjadi false. 

Baris 59-60 memastikan fungsi main() berjalan saat file dijalankan secara langsung.
